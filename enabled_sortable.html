Plugin enabled with support for sorting

If you sort the images and change their order in the DOM then you can run into problems. You can configure the plugin to be sortable, then you just need to trigger the "sort" custom JQuery event when this occurs and the plugin will do the rest. 

 <img data-original="img/example.jpg" src="img/grey.gif">
 $("img").lazyload({
     sortable: true
 });
 $(document).ready(
	function() {
		$("#teams").tablesorter().bind("sortEnd",function() { 
	        $(window).trigger("sort"); 
	    }); 
	}
);
